# homebrew-scrop

A [Homebrew](https://brew.sh) tap for [scrop](https://github.com/dfla-me/scrop),
a CLI that crops sub-images (photos, sticky notes, receipts) out of a scanned
composite image.

## Install

```sh
brew tap dfla-me/scrop
brew install scrop
```

## Uninstall

```sh
brew uninstall scrop
brew untap dfla-me/scrop
```

See the [main repo](https://github.com/dfla-me/scrop) for usage and source.
